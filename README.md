# computerProgrammingNTPU

This repository contains the programming assignments, labs, and projects for the **Computer Programming** course at National Taipei University (NTPU). The course focuses on building foundational programming skills and applying them to complex data-driven logic and predictive modeling.

## 🚀 Overview
The focus of this work is on implementing structured decision logic and leveraging scientific Python libraries to handle and analyze data.

## 📂 Repository Structure
* **/Assignments**: Implementation of logic-based systems and rule-based decision models.
* **/Labs**: Exercises focusing on scientific data handling (NumPy, Pandas).
* **/Project**: Capstone project involving high-accuracy predictive modeling and AI simulations.

## 🛠️ Tech Stack & Libraries
* **Language**: Python 3.x
* **Scientific Computing**: NumPy, Pandas
* **Visualization**: Matplotlib, Seaborn
* **Environment**: Jupyter Notebooks, VS Code

## 📝 Key Learning Modules
1.  **Programming Foundations**: Syntax, Data Types, and Operators.
2.  **Control Structures**: Automating logic using If/Else and Loops.
3.  **Scientific Data Handling**: Advanced array operations with NumPy.
4.  **Data Analysis & Visualization**: Processing datasets and visualizing trends.
5.  **AI Simulation**: Developing predictive models with high-accuracy alerts.

## 📊 Project Highlight: Decision Support System
Implemented a rule-based system that processes multi-parameter inputs to determine categorical outcomes. This was later extended with an AI simulation layer to flag high-alert cases based on predictive probability scores.

## 📖 Reference Material
* *Python for Data Analysis* by Wes McKinney.
* *Machine Learning and Python for Human Behavior* by M.Z. Uddin.
